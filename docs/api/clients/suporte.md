# API - Suporte

## Endpoint

| Método | Endpoint         | Descrição                  |
| ------ | ---------------- | -------------------------- |
| POST   | /api/v1/clients/enviarEmail | Envio de e-mail de suporte |

## Observações

- A rota recebe os dados do formulário de contato.
- O envio é realizado pelo backend utilizando Nodemailer.
