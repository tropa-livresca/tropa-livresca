Este é o projeto de TCC de uma editora fictícia.

Ele está codificado em Javascript.

Para modificá-lo, basta abrir o terminal e digitar os seguintes comandos:

git init git config --global user.email git config --global user.name git config --global user.password

git clone https://github.com/GabrielRD05092008/tropa-livresca.git

git remote add origin https://github.com/GabrielRD05092008/tropa-livresca.git

Em seguida, escrever:

npm install

Tanto dentro da pasta frontend, quanto do backend.

A cada mudança, digitar os comandos:

git checkout main git add . git status git push -u origin main

(Depois da primeira vez usando git push -u origin main, das próximas vezes é só usar git push)
