# 🗺️ Documentação do Modelo de Dados (Supabase/PostgreSQL)

Este documento contém o mapeamento estrutural (DDL) das tabelas do banco de dados do projeto. Ele serve como documentação técnica e script de recuperação para a arquitetura de tabelas.

---

## 🛠️ Instruções para Execução e Implantação

Se você precisar recriar estas tabelas em um banco de dados limpo ou em um novo ambiente do Supabase, siga rigorosamente as diretrizes abaixo:

1. **Dependências de Sequências (Sequences):**
   - A tabela `cupons` possui um vínculo com uma sequência de incremento automático (`DEFAULT nextval('cupons_2_id_seq'::regclass)`).
   - Certifique-se de que essa sequência exista ou altere o campo para `bigserial` / `generated always as identity` caso crie o banco do zero de forma puramente relacional.

2. **Chaves Estrangeiras pendentes:**
   - Este bloco de código contém estritamente a estrutura estrutural de criação (`CREATE TABLE`). Os comandos de amarração de chaves estrangeiras (`ALTER TABLE ... ADD CONSTRAINT`) devem ser executados apenas **após** a criação bem-sucedida de todas as tabelas listadas aqui.

3. **Esquema de Destino:**
   - Todas as entidades estão mapeadas explicitamente para o esquema `public` (`CREATE TABLE public.nome_da_tabela`).

## 🔗 Relacionamentos e Restrições (Chaves Estrangeiras)

As diretivas abaixo realizam as amarrações lógicas entre as entidades do ecossistema. Certifique-se de executar este bloco **apenas após** a criação de todas as tabelas listadas anteriormente.

```sql
-- ==========================================
-- CONSTRANGIMENTOS: adm_credenciais
-- ==========================================
ALTER TABLE public.adm_credenciais
    ADD CONSTRAINT fk_adm_credenciais_user FOREIGN KEY (fk_user_profile_id) REFERENCES public.users_profile(id) ON DELETE CASCADE;

-- ==========================================
-- CONSTRANGIMENTOS: blogs
-- ==========================================
ALTER TABLE public.blogs
    ADD CONSTRAINT fk_blogs_adm FOREIGN KEY (fk_adms_id) REFERENCES public.adm_credenciais(id) ON DELETE RESTRICT,
    ADD CONSTRAINT fk_blogs_categoria FOREIGN KEY (fk_categorias_id) REFERENCES public.categorias(id) ON DELETE SET NULL;

-- ==========================================
-- CONSTRANGIMENTOS: livros
-- ==========================================
ALTER TABLE public.livros
    ADD CONSTRAINT fk_livros_user FOREIGN KEY (fk_user_profile_id) REFERENCES public.users_profile(id) ON DELETE CASCADE,
    ADD CONSTRAINT fk_livros_categoria FOREIGN KEY (fk_categorias_id) REFERENCES public.categorias(id) ON DELETE SET NULL;

-- ==========================================
-- CONSTRANGIMENTOS: comentarios
-- ==========================================
ALTER TABLE public.comentarios
    ADD CONSTRAINT fk_comentarios_user FOREIGN KEY (fk_user_profile_id) REFERENCES public.users_profile(id) ON DELETE CASCADE,
    ADD CONSTRAINT fk_comentarios_livro FOREIGN KEY (fk_livros_id) REFERENCES public.livros(id) ON DELETE CASCADE,
    ADD CONSTRAINT fk_comentarios_blog FOREIGN KEY (fk_blog_id) REFERENCES public.blogs(id) ON DELETE CASCADE;

-- ==========================================
-- CONSTRANGIMENTOS: denuncias
-- ==========================================
ALTER TABLE public.denuncias
    ADD CONSTRAINT fk_denuncias_user FOREIGN KEY (fk_user_profile_id) REFERENCES public.users_profile(id) ON DELETE CASCADE,
    ADD CONSTRAINT fk_denuncias_categoria FOREIGN KEY (fk_categorias_id) REFERENCES public.categorias(id) ON DELETE CASCADE;

-- ==========================================
-- CONSTRANGIMENTOS: enderecos
-- ==========================================
ALTER TABLE public.enderecos
    ADD CONSTRAINT fk_enderecos_user FOREIGN KEY (fk_user_profile_id) REFERENCES public.users_profile(id) ON DELETE CASCADE;

-- ==========================================
-- CONSTRANGIMENTOS: usuarios_cartoes
-- ==========================================
ALTER TABLE public.usuarios_cartoes
    ADD CONSTRAINT fk_cartoes_user FOREIGN KEY (fk_user_profile_id) REFERENCES public.users_profile(id) ON DELETE CASCADE;

-- ==========================================
-- CONSTRANGIMENTOS: vendas
-- ==========================================
ALTER TABLE public.vendas
    ADD CONSTRAINT fk_vendas_user FOREIGN KEY (fk_user_profile_id) REFERENCES public.users_profile(id) ON DELETE RESTRICT,
    ADD CONSTRAINT fk_vendas_met_pagamento FOREIGN KEY (fk_met_pagamentos_id) REFERENCES public.met_pagamentos(id) ON DELETE RESTRICT;

-- ==========================================
-- CONSTRANGIMENTOS: itens_venda
-- ==========================================
ALTER TABLE public.itens_venda
    ADD CONSTRAINT fk_itens_venda_pai FOREIGN KEY (fk_vendas_id) REFERENCES public.vendas(id) ON DELETE CASCADE,
    ADD CONSTRAINT fk_itens_venda_livro FOREIGN KEY (fk_livros_itens_id) REFERENCES public.livros(id) ON DELETE RESTRICT;

-- ==========================================
-- CONSTRANGIMENTOS: cupons
-- ==========================================
ALTER TABLE public.cupons
    ADD CONSTRAINT fk_cupons_livro FOREIGN KEY (fk_livros_id) REFERENCES public.livros(id) ON DELETE CASCADE,
    ADD CONSTRAINT fk_cupons_adm FOREIGN KEY (fk_adm_credenciais_id) REFERENCES public.adm_credenciais(id) ON DELETE SET NULL;

-- ==========================================
-- CONSTRANGIMENTOS: revisoes
-- ==========================================
ALTER TABLE public.revisoes
    ADD CONSTRAINT fk_revisoes_user FOREIGN KEY (fk_user_profile_id) REFERENCES public.users_profile(id) ON DELETE CASCADE,
    ADD CONSTRAINT fk_revisoes_livro FOREIGN KEY (fk_livro_id) REFERENCES public.livros(id) ON DELETE CASCADE;

-- ==========================================
-- CONSTRANGIMENTOS: notificacoes
-- ==========================================
ALTER TABLE public.notificacoes
    ADD CONSTRAINT fk_notificacoes_user FOREIGN KEY (fk_user_profile_id) REFERENCES public.users_profile(id) ON DELETE CASCADE;

-- ==========================================
-- CONSTRANGIMENTOS: notificacoes_lidas
-- ==========================================
ALTER TABLE public.notificacoes_lidas
    ADD CONSTRAINT fk_lidas_notificacao FOREIGN KEY (fk_notificacao_id) REFERENCES public.notificacoes(id) ON DELETE CASCADE,
    ADD CONSTRAINT fk_lidas_user FOREIGN KEY (fk_user_profile_id) REFERENCES public.users_profile(id) ON DELETE CASCADE;
```
